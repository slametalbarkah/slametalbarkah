Nullsoft Scriptable Install System

Nullsoft Scriptable Install System (NSIS) adalah sistem instalasi skrip-driven untuk Microsoft Windows dengan minimal overhead yang didukung oleh Nullsoft, pencipta Winamp. NSIS telah menjadi alternatif yang banyak digunakan untuk produk komersial dan proprietary seperti InstallShield, dengan pengguna termasuk Amazon, Dropbox, Ubisoft, BitTorrent, dan McAfee.

NSIS adalah perangkat lunak gratis yang dirilis dibawah kombinasi lisensi perangkat lunak bebas, terutama lisensi zlib.
