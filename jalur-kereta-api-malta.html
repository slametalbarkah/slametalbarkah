Jalur kereta api Malta

Jalur kereta api Malta adalah satu-satunya jalur kereta api yang pernah ada di pulau Malta dan terdiri dari satu jalur yang menghubungkan Valletta dengan Mdina. Jalur ini beroperasi dari tahun 1883 hingga 1931 dan dalam bahasa Malta dikenal dengan sebutan il-vapur tal-art (kapal darat).

Usulan pembangunan jalur kereta api di Malta dicetuskan untuk pertama kalinya pada tahun 1870 oleh J. S. Tucker. Alasannya adalah untuk menghubungkan ibu kota Malta di Valletta dengan bekas ibu kota di Mdina agar lama perjalanan di antara kedua kota tersebut dapat terpangkas dari tiga jam menjadi kurang dari setengah jam. Awalnya rancangan yang diusulkan adalah rancangan yang dibuat oleh John Barraclough Fell. Pada tahun 1879, rancangan ini diganti dengan rancangan perusahaan Wells-Owen & Elwes, London.

Akibat permasalahan dengan akuisisi lahan untuk membangun jalur kereta api, pembangunan memakan waktu lebih lama dari yang diperkirakan. Jalur ini akhirnya dibuka pada tanggal 28 Februari 1883 pada pukul 15:00, ketika kereta pertama meninggalkan Valletta dan tiba di Mdina setelah sekitar 25 menit.





== Penutupan ==

Bus pertama di Malta diperkenalkan pada tahun 1905 dan menjadi populer pada tahun 1920-an, sehingga mengakibatkan kemunduran industri kereta api di Malta. Jalur kereta api ini akhirnya berhenti beroperasi pada tanggal 31 Maret 1931.
